# Deployer Recipes

This repository contains UBC CS department recipes to integrate with deployer.

## Installing

~~~sh
composer require ubc-cpsc/deployer-recipes --dev
~~~

Include recipes in `deploy.php` file.

```php
require 'recipes/drupal8.php';
```
